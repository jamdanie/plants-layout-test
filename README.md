# UI Layout Lab

A pure HTML/CSS layout study focused on structure, spacing, layering, and visual hierarchy.

This project explores how far clean layout and motion cues can go **without JavaScript or frameworks**.

## Live Demo
https://jamdanie.github.io/ui-layout-lab/

## What this project demonstrates
- Centered, responsive layout structure
- Visual hierarchy through spacing and typography
- Gradient layering and depth
- Subtle motion using CSS only
- Clean composition without JavaScript

## Constraints
- No JavaScript
- No frameworks
- HTML + CSS only

## Purpose
This project was built as a layout exercise to strengthen fundamentals before adding interactivity or frameworks.
